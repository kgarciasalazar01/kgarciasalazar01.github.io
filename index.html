Caso de estudio: Alke Wallet

Desarrollé una aplicación web tipo Wallet utilizando Python, Flask y Bootstrap. El objetivo era permitir a los usuarios registrar, visualizar y gestionar sus movimientos financieros. La aplicación permite ingresar ingresos y gastos, ver el balance actualizado y revisar el historial de transacciones.

Este proyecto me permitió comprender cómo se estructura una aplicación web desde cero, separar correctamente la lógica del backend de la interfaz, manejar rutas, formularios y procesamiento de datos, y mejorar mi pensamiento lógico.
